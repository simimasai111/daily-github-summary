# 每日 GitHub 项目汇总

## 项目概述

通过 GitHub Actions 定时抓取 GitHub Trending 热门项目，调用 AI 进行摘要总结，最终通过 Server 酱 推送到手机端。

## 技术栈

- Python (requests)
- GitHub Actions (CI/CD)
- OpenAI 兼容 API
- Server 酱 推送接口

## 核心特性

- 每日自动抓取 GitHub Trending Top 10 项目
- 支持自定义 AI API（Base URL / Key / Model）
- AI 生成中文项目总结
- Server 酱 Markdown 格式推送
- 支持手动触发工作流

## 目录结构

```
.
├── .github/workflows/
│   └── daily-summary.yml    # GitHub Actions 定时任务
├── src/
│   ├── config.py            # 配置读取
│   ├── fetcher.py           # 抓取 Trending
│   ├── summarizer.py        # AI 总结
│   ├── formatter.py         # Markdown 格式化
│   ├── pusher.py            # Server 酱 推送
│   └── main.py              # 主入口
├── requirements.txt
└── README.md
```
